# calcula-vet-web

## O Problema
O Brasil é o país com o maior número de cursos de Medicina Veterinária do mundo — são cerca de 580 faculdades, que formam aproximadamente 60 mil veterinários por ano. Aliado a isto, a quantidade de pets no Brasil, uma das mariores do planeta, segue em crescimento. Fica claro que o ramo veterinário é um mercado crescente e altamente competitivo onde empresas disputam espaço nas prescrições médico-veterinárias.
No entanto, por trás desse crescimento, existe um problema silencioso que compromete a conversão da prescrição em venda: o afastamento entre o conhecimento do veterinario e a realidade comercial.
Na prática, o veterinário sabe qual princípio ativo indicar — mas frequentemente não sabe quais apresentações comerciais estão disponíveis no mercado para aquele composto, em qual concentração, volume ou forma farmacêutica. Com centenas de produtos lançados anualmente, é humanamente impossível manter esse repertório atualizado.
Some a isso outra tarefa de rotina: o cálculo de dose. Cada paciente tem espécie, peso e particularidades próprias, e transformar a dose em uma prescrição prática requer tempo, atenção e, muitas vezes, consultas manuais — um processo sujeito a erros e ineficiências.
Até hoje, não existe uma plataforma que una esses dois mundos: que calcule automaticamente a dose para o paciente e, com base nesse cálculo, consulte um banco de dados de apresentações comerciais, filtrando e sugerindo apenas as opções compatíveis com aquela prescrição específica.
Calcula Vet Web nasce para preencher exatamente essa lacuna.

## Descrição
Projeto em Python 3 para cálculo de doses para veterinárias e sugestão de apresentações comerciais dos princípios ativos. 

## Objetivo
Criar uma calculadora de dose veterinária simples (espécie, peso, contexto de uso, via de administração) e cruzar com um banco de dados das apresentações comercias disponíveis, afunilando a resultados coerentes ao resultado da calculadora.

## Observações

### Extensão
Atualmente o Calcula Vet Web tem intenção de calcular fármacos, porém futuramente podem ser inclusas rações de prescrição.

### Modelo de Negócio
Além de resolver um problema real e urgente, o Calcula Vet Web possui potencial de monetização estruturado em duas frentes complementares:

B2B — Indústria farmacêutica e distribuidores: empresas pagam uma taxa periódica de manutenção para ter seus produtos indexados, atualizados e visíveis na plataforma no momento exato da prescrição — posicionamento estratégico de alto valor.
B2C — Veterinários: profissionais acessam a plataforma mediante assinatura, obtendo uma ferramenta que otimiza sua rotina clínica, reduz erros e agrega credibilidade à prescrição.
