For creating a new drug function, always follow the statement order described bellow:

- In case there's a species difference, ask for species. Then;
- In case there's a case or use difference, ask for case/use. Then;
- In case there's a dosage range to choose, ask the dosage. Then;
- Print "Administrar {dose:.2f}mg {via} {frequencia}."

Some criteria that must be taken in consideration:
- Always show the dose in mg.
- Always use two decimals for the dose.
- NEVER invent dose values that is not on guia_terapeutico_veterinario.md file which is the base-material.
- All drugs must be organized in an alphabetic order.
